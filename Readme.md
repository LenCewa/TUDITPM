## Branchstruktur

Die folgenden Branches sollen genutzt werden:
  - develop 
  - kafka
    - kafka/master
    - kafka/feature
      - kafka/feature/anbindung - z.B. erste Datenanbindung
  - spark
    - spark/master
    - spark/feauture
      - spark/feature/analytics - z.B. textAnalytics
